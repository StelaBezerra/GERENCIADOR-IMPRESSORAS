# Sistema de Gestão de Impressoras

Sistema web desenvolvido para centralizar, organizar e facilitar a consulta das informações técnicas das impressoras utilizadas nas unidades da empresa, apoiando a rotina do suporte de TI e a gestão de ativos.

---

## Visão Geral

A aplicação permite visualizar e consultar dados das impressoras de forma rápida e organizada, reduzindo o tempo gasto na busca por informações como modelo, IP, status e suprimentos.
O projeto foi desenvolvido com foco em uso interno, simplicidade e eficiência operacional.

---

## Tecnologias Utilizadas

* **HTML5** — estrutura da aplicação
* **CSS3** — estilização e layout responsivo
* **JavaScript** — lógica de funcionamento
* **XLSX.js** — exportação de relatórios em formato Excel

---

## Funcionalidades

* Consulta de impressoras por **loja**
* Busca rápida por modelo, IP ou informações gerais
* Exportação de dados em **XLSX**
* Cópia rápida de IP e dados relevantes
* Interface simples, intuitiva e otimizada para uso interno
